My project is an advanced file management system developed using CakePHP. 
It provides functionalities similar to Google Drive, allowing users to upload, download, view, edit, and delete files and folders through a user-friendly interface.
The system supports full CRUD (Create, Read, Update, Delete) operations for efficient file organization. 
An administration panel is included to manage users, permissions, and system resources, ensuring secure access control and effective management of the platform.
The project aims to offer a reliable and centralized solution for storing and managing digital documents.
